# Screenshots

公开截图请统一放在这个目录，并使用脱敏后的演示配置生成。

## 命名建议

- `lobster-canvas.png`
- `workspace-files.png`
- `backups-json.png`
- `guide.png`

## 脱敏要求

- 不要出现真实 token、secret、appId、clientSecret
- 不要出现真实用户 ID、群 ID、会话 ID
- 不要出现真实工作区原文
- 不要出现带个人用户名的绝对路径
- 不要出现本机浏览器书签、系统通知、任务栏隐私信息

## 推荐顺序

1. 龙虾画布
2. 工作区文件
3. 备份 / JSON
4. 原理导览

## README 中的目标位置

截图补齐后，可以在仓库 README 的“预览”部分按这个顺序插入图片。
